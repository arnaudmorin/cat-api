# cat-api

Provide some cat or dog gif


# Usage

Eventually, build the app

```bash
sudo ./build.sh
```

Then, run the app

```bash
sudo ./run.sh
```

or use docker compose:

```bash
docker compose up -d
```

# Enjoy
Enjoy by browsing:

- For cats: [http://127.0.0.1:8080/cat](http://127.0.0.1:8080/cat)
- For dogs: [http://127.0.0.1:8080/dog](http://127.0.0.1:8080/dog)
